# Quiz Battle

Интерактивная игра-викторина "Quiz Battle" с системой ролей, чатом и уведомлениями.
Проект разработан в рамках практических работ №3-6.

## Технологии
- React (Vite)
- TypeScript
- CSS (Modern Monotone UI)
- LocalStorage (Mock DB)
- Lucide React (Icons)

## Функционал
1. **Ролевая система:** Админ, Игрок, Гость.
2. **База вопросов:** Добавление, редактирование и удаление вопросов (доступно только администратору).
3. **Игровой процесс:** Два игрока соревнуются, отвечая на вопросы.
4. **Таблица лидеров:** Сохранение истории побед и серий.
5. **Чат и Уведомления:** Реалтайм-обмен сообщениями между пользователями и автоматические системные уведомления о результатах игры.
6. **ER-диаграмма:** Интерактивное визуальное представление структуры базы данных.

## Запуск проекта
```bash
npm install
npm run dev
```

## Сборка (Build)
```bash
npm run build
```
